# Alfido Tech Intern Project

This repository contains cybersecurity practical tasks completed as part of the Alfido Tech Internship Program.  
All tasks were performed in controlled and authorized environments for educational purposes.

## Tasks Included
1. Reconnaissance & Vulnerability Scanning  
2. Wireshark Packet Analysis  
3. Incident Response Simulation  

## Tools & Technologies Used
- Nmap
- Nikto
- Wireshark
- Windows Security Logs

## Repository Structure
Each task directory contains:
- A task-specific README
- Detailed analysis reports
- Screenshot placeholders for evidence

⚠️ Disclaimer:  
This project is strictly for learning and ethical cybersecurity practice.
